#Array

  • Array vandhu oru periya container mathri adhukulla namba variables lam potutu aprm refer panikalam
  • Array ellam 0 based
  • Array oru primitive value adhunala array ulla iruka variable la const keyword use panirundhalum namba adha change panikalam
  • Array ulla oru statement use pana mudiyadhu
  • An Array is like a big container into which we can throw variables and then later reference them.
  • Array are 0 based
  • Array is not a primitive value so we can change the element inside an array even if we using const Keyword.
  • We should not write a statement inside an array.

#Operators

Push: (Add Element)

  • Array oda end la element ah add panum
  • Push method adds elements to the end of an array.

Unshift : (Add Element)

  • Array oda starting la element ah add panum
  • Adds element to the beginning of the array.

Pop : (Remove Element)

  • Push ku opposite
  • Opposite to Push.

Shift: (Remove Element)

  • Unshift ku opposite
  • Opposite to unshift.
