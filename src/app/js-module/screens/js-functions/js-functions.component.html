# Function Declaration:

This is the most common way to define a function.

# Function Expression:

Functions can also be defined as expressions and assigned to variables. This way of defining a function is known as a function expression.

# Arrow Function:

Arrow functions provide a concise syntax for writing function expressions. They use the => syntax and are often used for short functions.

# Calling a Function:

To execute a function, you call it by its name followed by parentheses, passing any required arguments.

# Parameters and Arguments:

Parameters are placeholders for values that the function uses.

Arguments are the actual values passed to the function when it is called.

# Returning a Value:

A function can return a value using the return statement. If no return statement is used, the function returns undefined.
