The if, else if, and else statements are used for conditional execution of code blocks.
They allow you to execute different blocks of code based on different conditions.

If

  • The if statement evaluates a condition.
  • If the condition evaluates to true, the code block inside the if statement is executed.
  • If the condition evaluates to false, the code block inside the if statement is skipped.

If Else

If the 'If' condition is false, the code block inside the 'else' statement is executed.

Else If

If the 'If' condition is false, the code block inside the 'else if' statement is executed. If both the 'if' & 'else if' is false 'else' will be executed.
