Basic Usage

  • The prompt() function displays a dialog box with a specified message, along with a text input field.
  • It returns the text entered by the user or null if the user cancels the input.

Adding Default Value

You can provide a default value for the input field by passing a second argument to the prompt() function.

Limitations of prompt()

  • The prompt() function only allows a single input field.
  • It cannot be customized to include multiple input fields directly.
