# Explanation

  1. Expression: idhu oru expression ah oru time dha run panum.
  2. Cases: Expression oda value oru oru case oda value la um compare pani run panum.
  3. Break: Switch block ah vitu veliya vara use aagum. Break illana condition true aana kuda adhu next iruka ella case ah um run panum
  4. Default: Idhu venum na use panikalam endha case um true aagala apdina default case run aagum
  • It is used to execute one block of code out of multiple options based on the value of an expression.
  • It is an alternative to using multiple if...else if statements and can make your code more readable when there are many conditions to check.

# Explanation

  1. Expression: The switch statement evaluates an expression once.
  2. Cases: The expression's value is compared with the values of each case clause.
  3. Break: The break statement exits the switch block. Without break, the code will continue to execute the next case even if a match has been found (fall-through behavior).
  4. Default: The default clause is optional and executes if none of the case values match the expression. It's like the else in if...else statements.
