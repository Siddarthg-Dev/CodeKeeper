#Template Literals

Nariya piece ah seathu oru string create panum

A template literal can assemble multiple pieces into one final string.

1. String Concatenation:

Idhu munadila irundhu use panura method + use pani string aprm variable onu seathu kamipanga

This is the traditional method of combining strings and variables using the + operator.

2. Template Literals:

Idhu latest method ES6 la kondu vandhanga idhu expressions ah backticks ` aprm ${{ veachi string an kamikum

This is the modern method introduced in ES6 that allows embedding expressions inside a string using backticks ` and ${{ syntax.
